# Elicitation Phase Checklist

## Functional Requirements
- [ ] Core features identified
- [ ] Feature descriptions/priorities
- [ ] User tasks/workflows mapped
- [ ] Inputs/outputs identified
- [ ] Required integrations listed

## Non-Functional Requirements
**Performance:** Response time, throughput, concurrent users
**Security:** Auth, encryption, audit logging
**Reliability:** Availability, RTO/RPO, backup
**Usability:** Accessibility, browsers, devices, locales
**Scalability:** Scaling approach, growth projections
**Compliance:** Regulatory mapped to NFRs

## Constraints
- [ ] Technical constraints
- [ ] Business constraints (budget, timeline)
- [ ] Resource/infrastructure constraints
- [ ] Integration constraints

## Risks
- [ ] Technical/business/schedule risks
- [ ] Probability and impact assessed
- [ ] Mitigation strategies defined

## Assumptions & Dependencies
- [ ] Documented with validation plan
- [ ] Owners assigned, status tracked

## Phase Gate
- [ ] All activities complete
- [ ] Requirements reviewed by stakeholders
- [ ] Ready for Specification
