# Discovery Phase Checklist

## Domain Analysis
- [ ] Industry/vertical identified
- [ ] Regulatory requirements (if any)
- [ ] Competitors researched
- [ ] Domain terminology documented
- [ ] Common patterns identified

## Stakeholder Mapping
- [ ] Primary/secondary users identified
- [ ] Business/technical stakeholders
- [ ] External stakeholders (if any)
- [ ] RACI matrix drafted

## Vision & Goals
- [ ] Problem statement articulated
- [ ] Vision statement (one sentence)
- [ ] Success metrics (KPIs)
- [ ] 6-month/1-year success criteria
- [ ] In-scope/out-of-scope defined

## Documentation
- [ ] Discovery Worksheet completed
- [ ] Key findings summarized
- [ ] Open questions documented
- [ ] Assumptions documented

## Phase Gate
- [ ] All activities complete
- [ ] Stakeholders reviewed
- [ ] No blocking questions
- [ ] Ready for Elicitation
