# Agile-Driven Development Framework - Templates
**Version:** v0.9.0
**Module:** Templates

---

**Note:** Backlogs stored as GitHub issues, not local files. Use `Create-Backlog`.

## Story Format
```markdown
### Story [ID]: [Title]
As a [user] I want [goal] So that [benefit]
Acceptance Criteria: [checkboxes]
Story Points: [1,2,3,5,8,13,21]
Priority: [High/Medium/Low]
Status: [Backlog/Selected/In Progress/Done]
```

## Sprint Summary Format
```markdown
# Sprint [N] Summary
Sprint Goal: [goal]
Duration: [dates]
Completed: [X] story points
Velocity: Planned vs Completed
Key Achievements: [list]
Challenges: [list]
New Stories Discovered: [list]
```

## Retrospective Format
```markdown
# Sprint [N] Retrospective
What Went Well: [list]
What Could Improve: [list]
Action Items: [checkboxes]
Velocity Trends: [history]
```
